Meetup.com for WordPress allows you to integrate your Meetup.com account with your website to bring in meet up and event information.

== Description ==

Meetup.com for WordPress fetches event information from Meetup.com and creates posts for each event. Additionally, Meetup.com for WordPress provides an event calendar for displaying upcoming events either on a page or as a widget.

Uses the [Meetup PHP API](https://github.com/wizonesolutions/meetup_api) developed by [wizonesolutions](http://www.wizonesolutions.com).

== Installation ==

1. Upload `meetup-for-wordpress.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the options page located at Settings -> Meetup.com and follow the instructions.
1. To include the event calendar on a page, create a new page and in the content editor, enter `[wp-meetup-calendar]` where you want the calendar to appear on the page.
